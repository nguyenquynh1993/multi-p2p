vic: What's New?

October 16, 1995

  • Added support for the Matrox Metero Video adaptor, a PCI-based capture board, thanks to Jim Lowe. A driver is available (in source form) for FreeBSD. See ftp://ftp.cs.uwm.edu/pub/FreeBSD/meteor*.

October 16, 1995

  • Released the a26 snapshot, which fixes major foul-ups in a25 snapshot (i.e., missing files, major bugs, bad URLs).
  • Announced a25 snapshot on the rem-conf@es.net mailing list.

October 14, 1995

  • Added a page about on-going research projects.
  • Released the a25 snapshot.

October 4, 1995

  • The next version of vic (2.7) implements the July 1995 RTP/H.261 Payload Format specification. This change precludes backward compatibility with the vic-2.6 H.261 implementation. v2.7 is currently in alpha test.
  • Moved the alpha-test directory to a public area.
  • Created html documents from the README and CHANGES files.
